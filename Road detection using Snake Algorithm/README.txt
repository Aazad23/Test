-----------------------------------------------------------------------
Setup
-----------------------------------------------------------------------
1. Make sure your default python installation is Python 3
2. Install virtualenv:
        pip install virtualenv
3. Create a virtualenv named venv:
        virtualenv venv
4. Activate the virtualenv:
        source ./venv/bin/activate {for linux/osx}
        .\venv\Scripts\activate {for win}
5. Install dependencies in virtualenv:
        pip install -r requirements.txt


-----------------------------------------------------------------------
Run
-----------------------------------------------------------------------
python app.py


-----------------------------------------------------------------------
Working
-----------------------------------------------------------------------
1. Select an image from the file picker
2. Set parameters for snake
3. Click on Process Image button
